# Expense Tracker

## Project Description

Expense Tracker is a Python-based command-line application that helps users manage their daily expenses efficiently. Users can add, view, update, delete, and search expenses while maintaining records in a JSON file. The application also provides expense summaries to help users track their spending habits.

---

## Features

* Add new expenses
* View all expenses
* Update existing expenses
* Delete expenses
* Search expenses by category
* Generate expense summaries
* Store data permanently using JSON
* Input validation and error handling
* Menu-driven interface

## How to Run the Project

### Prerequisites

* Python 3 installed on your system

### Steps

1. Clone the repository

   git clone <repository-url>

2. Navigate to the project directory

   cd ExpenseTracker

3. Run the application

   python main.py
